# ilico.org

ilico.org - new webiste made with typescript + vite + threejs + tailwindcss.
Previous website at https://www.ilico.org/

## npm workspaces

-   app (app)
-   packages/\*
    -   components (pkg-components)
    -   utils (pkg-utils)
    -   webgl (pkg-webgl)

## scripts

### development

```
npm i
npm start
```

### preview

```
npm run build
npm run preview
```
